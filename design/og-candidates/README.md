## Shipped artifact

`01-winner-clean-editorial.png` is the source that **is shipped** as `public/og-image.jpg` (1200×630 JPEG, ~85% quality, ~113 KB). The conversion is performed by `scripts/generate-og-image.mjs` (npm run `og:generate`); see the repo-root `README.md` "Brand & SEO" section for usage.

The original AI-generated PNGs are kept here in source quality. The shipped `public/og-image.jpg` is a derived artifact (resized + recompressed) and should not be edited in place — re-derive it from the winner PNG (or a future replacement) by running `npm run og:generate` (default mode reads `01-winner-clean-editorial.png`; pass `--source <path>` to point at a different candidate).

If the photo source is ever lost, `npm run og:generate -- --bridge` regenerates a typographic fallback (cream card + "Mukyala" wordmark + tagline) so the OG slot never goes blank.
